# TAZROUT ESP32 Sensor Module

IoT Irrigation System - ESP32 Sensor & Network Communication Layer

## Project Overview

This module is part of the TAZROUT IoT Irrigation System, responsible for:
- Real-time soil moisture monitoring
- Environmental condition tracking (temperature & humidity)
- WiFi and MQTT communication
- Remote command processing
- Data validation and logging
